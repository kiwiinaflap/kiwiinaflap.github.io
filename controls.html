Diagram

The diagram is drawn to scale for the following input parameters:

  • Servo vertical position
  • Servo arm radius BC
  • Servo arm angle
  • Servo centre offset
  • Control horn radius AD
  • Control horn angle

The diagram is NOT drawn to scale for the following input parameters:

  • Control surface hinge to servo arm AB (servo moves, but not to scale)
  • Control surface width AE (remains fixed)

All calculations are made according to parameter input settings, not to the drawing. This ensures all output information is accurate, regardless of the drawing.

Inevitably, there will be some combination of settings that will yield peculiar or bizzare results. Such settings are unlikely to be 'good' solutions to a well tuned setup and should be avoided. Fortunately, they will not break the simulation.

Graph

The graph shows how the angle of the control surface responds to the relative angle of the servo arm over a full sweep, typically from -50 to +50 deg. The angle of the control surface is plotted on the vertical Y axis (deg) against the relative angle of the servo arm on the horizontal X axis (deg).

Sweep

Use the 'Sweep' slider to manipulate the servo arm through its full range. The play ▶ button automates this function.

Input Parameters

Click or touch any input value to reveal the Control Panel. The panel can be dragged to a new location if desired. Use its slider or ◀ ▶ buttons to adjust the selected value. Move up or down the parameter list with the navigation ▲ ▼ buttons.

The servo can also be dragged to any location within the range of 'Control surface hinge to servo arm AB' and 'Servo vertical position'. The drag function includes fine stepping to enable precise positioning with small mouse movements. This function is disabled for mobile phones.

Control surface hinge to servo arm AB

Adjust the length between the control surface hinge point A and servo arm's pivot point B. Range 40 to 400 mm. The effect on control deflections is minimal beyond this range.

The diagram does not display the length AB to scale. The graph is not affected by this as all calculations use the parameter length AB, not the diagram's.

Servo vertical position

Move the servo up or down relative to the control surface datum line (horizontal line through point A). Range -25 to +25 mm.

Servo arm radius BC

Set the radius of the servo arm, length BC. Range 3 to 30 mm. Try to minimise the radius to maximise torque output, increasing it only as much as necessary to achieve the desired control surface range (plus margin) and to provide clearance for the connecting rod clevis.

Servo arm angle

Set the servo arm angle relative to the datum line. With reference to the diagram, an angle of 0 degrees is along the datum line to the right. From this datum, positive angles rotate the arm anti-clockwise and negative angles, clockwise. Range -150 to +150 degrees.

Servo centre offset

Shift the centre point of the servo to create unequal movement either side of centre. The offset is added to the normal sweep (-50 to +50 deg), eg. a 30 deg offset will cause a sweep of -20 to +80 deg. Offset provides mechanical differential, typically required for the setup of flaps, spoilers, retracts, etc.

Control horn radius AD

Set the radius of the control horn, length AD. Range 4 to 40 mm. If you have a choice, set the radius so that the control surface moves a little beyond the required range (plus margin).

Control horn angle

Set the control horn angle relative to the datum line. With reference to the diagram, an angle of 0 degrees is along the datum line to the right. From this datum, positive angles are measured anti-clockwise and negative angles, clockwise. Range -135 to -45 and +45 to +135 degrees.

Control surface width AE

Range 10 to 300 mm. Set the control surface width to see the corresponding maximum positive and negative control surface deflections, measured from point E. Should your control surface width lie outside the range, or if you wish to improve the resolution near the low end of the range, scale it: scale the width by some factor (eg x10) and divide the displayed deflection values by the same factor.

Connecting rod length CD

This is the calculated length of the connecting rod from point C to point D.
